== Description ==

Import Simple Tagging tags into WordPress tags.

== Installation ==

1. Upload the `stp-importer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Tools -> Import screen, Click on Simple Tagging
